# IP Finder #
IP Finder - это программа на python, которая ищет минимальную маску между двумя случайными IP-адресами. Он поддерживает IPv4 и IPv6.

Для установки введите команды:

```
git clone https://github.com/Mager2112/IP-Finder.git
cd IP-Finder
```

## Теория ##
> [!NOTE]
> **Маска подсети** (или CIDR-префикс) определяет, сколько старших бит IP-адреса являются общими для всех устройств в подсети. Чем больше число после `/`, тем меньше адресов в подсети.

### Как работает алгоритм ###
Программа преобразует IP-адреса в двоичный вид и сравнивает их бит за битом, начиная со старшего. Результат — количество совпадающих бит подряд (самый длинный общий префикс).

> [!TIP]
> Программа находит **максимально возможную** маску (самую специфичную подсеть), которая объединяет оба адреса. Например, для адресов `192.168.1.1` и `192.168.1.100` результат будет `/25`, а не `/24`, потому что `/25` — более точная подсеть, все еще содержащая оба адреса.


## Использование ##
Для использования нужно ввести в строку следующие команды:
```
python3 ip_finder.py <IP1> <IP2>
```
Для работы можно использовать как IP-адреса как версии IPv4, так и IPv6. Образец **IPv4** - 192.168.10.1, **IPv6** - 2001:0db8:0000:0000:0000:0000:0000:0001 или сокращённо 2001:db8::1

> [!EXAMPLE]
> **Примеры работы программы:**
> - IPv4: `192.168.1.1` и `192.168.1.100` → `/25` (255.255.255.128)
> - IPv4: `192.168.1.1` и `192.168.2.1` → `/22` (255.255.252.0)
> - IPv6: `2001:db8::1` и `2001:db8::2` → `/126`
> - IPv6: `2001:db8:1234::1` и `2001:db8:1235::1` → `/47`
